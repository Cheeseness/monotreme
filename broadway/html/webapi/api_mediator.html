Mediator object documentation
 
Mediator()
creates a mediator object to interact with
Return null
NOTE use the new method Example: var mediator = new Mediator()
Methods
 
set_target(hostname)
hostname of the machine you want to interact with
hostname the hostname of the machine you want to interact with
ExampleExample:bob_m
Return null
 
 
get_target()
the hostname currently being used
Return string
 
 
get_node(path)
returns a node object
path node path
ExampleExample:/services/time
Return a Node object
 
 
get_nodes(nodes)
returns a nodes object
nodes array of node path
ExampleExample:['/services/time,'/interfaces/com1...,etc]
Return nodes object
 
 
login(username,password)
logs into the host that is serving the webpages
username username on the mediator that the webpage is being server from, not the target machine
password users password
Return a string that is the session
NOTE The username is a user on the machine that the webpage is being served from, NOT the target machine
 
 
logout()
logs out of the current host
Return null
