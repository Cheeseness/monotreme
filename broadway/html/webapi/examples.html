3 simple steps to help get you started...
Step 1: Enable the webdev node tags
Step 2: Design and create a custom HTML file including some special attributes in specified HTML tags
Step 3: Include some Javascript code in your page
Step 4: Publish the files to the server
< previous step | next step >
Step 1

In the configuration tool find the webdev node under /services/netowork
check "enable" and "Accept FTP connections"

Click image or here to enlarge

Save the configuration
Step 2

Design and create an HTML file using your favorite HTML editor, like Dreamweaver™
Place a node attribute in certain HTML tag elements. The node attribute will contain the node path of the node value you would like to display.
Example: <span node="/services/time"></span> or <td node="/interfaces/relay1"></td>
Important Note: always use the closing tag "</span> or </td>"

Supported tags:
<span>
<div>
<td>
<img>
<input>
Important Note: <img> needs a couple of attributes added to the tag to work.
Step 3

Include the following javascript code in between <title> tag of your document
<script language="javascript"src="js/sdk2.js"></script>
<script language="javascript">
    m = new Mediator();
    page_properties(m,5000);
    start_refresh_loop(page_properties);
</script>

The above example will ask you to login the first time it tries to refresh and then continue to refresh every 5 seconds
Step 4

ftp the file of files to the server. Use webdev as the username and the same as the default password. To learn more about pubishing files to the host machine, visit the Publishing help page
