Publishing files to a mediator is very easy:

Step 1

   
Make sure that the Enabled and Accepting FTP connections check boxes in the webdev node are checked.

Example:
   
Enabled 
Accepting FTP connections 
   
Click image or here to enlarge

Step 2
   
FTP the files to the Mediator. Most HTML editors (including Dreamweaver) have a "site" configuration, which allows you to FTP files straight from the editor to a host (in this case, to the Mediator). 
   
The default username/password is webdev/webdev
