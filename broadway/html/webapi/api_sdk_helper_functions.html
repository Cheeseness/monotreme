login_window(get_mediator,callback)
opens a login window so the user can login
get_mediator name of function to call to get a refrence to the mediator object
callback name of the callback function you want called after login
Return null
NOTE get_mediator and callback are strings to functions in your main page the login window uses the eval function to call your function
NOTE Example: eval('window.opener.' + get_mediator + '()')
NOTE get_mediator should return a refrence to your mediator object
NOTE the callback function will be passed the mediator object that is logged in
 
precision(element,value,decimal_places,display_error)
takes a float value and returns a float to the decimal places specified and then updates the given element
element refrence to an HTML element
value current value
decimal_places the number of decimal_places to change to
display_error value to display if there is an error
Return null
